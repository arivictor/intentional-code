They say write what you needed to know when you were learning. The same goes for learning to code. In this course we'll build our own API framework from scratch. We'll apply the patterns and principles of intentional code to the framework itself, so you understand not just how to use it, but why it works the way it does. By the end, you'll have a working HTTP framework you built yourself, and a deep understanding of the conventions every team ends up writing for itself. From there you can go on to extend the framework with your own features, and at worst have a greater appreciation for the one you choose to use in production.
